## List Cameras

<ParamField path="method" type="GET">
  GET /api/cameras
</ParamField>

Enumerate all cameras connected via USB and network.

<CodeGroup>

```bash Request
curl http://localhost:8080/api/cameras
```


```json Response
{
  "success": true,
  "message": "Camera discovery completed",
  "cameras": [
    {
      "id": "D10F60149B0C",
      "model": "ILCE-9M3",
      "connectionType": "USB",
      "connected": false
    }
  ]
}
```

</CodeGroup>

<ResponseField name="cameras" type="array">
  <Expandable title="Camera object">
    <ResponseField name="id" type="string">
      Unique camera identifier
    </ResponseField>
     <ResponseField name="model" type="string">
      Camera model name (e.g. ILCE-9M3)
    </ResponseField>
     <ResponseField name="connectionType" type="string">
      USB or Network
    </ResponseField>
     <ResponseField name="connected" type="boolean">
      Whether camera is currently connected
    </ResponseField>
  </Expandable>
</ResponseField>

## Connect Camera

<ParamField path="method" type="POST">
  POST /api/cameras/{cameraId}/connection
</ParamField>

Establish connection in the specified mode.

<ParamField path="cameraId" type="string" required>
  Camera ID from discovery
</ParamField>

<ParamField body="mode" default="remote" type="string">
  Connection mode: `remote`, `remote-transfer`, or `contents-transfer`
</ParamField>

<ParamField body="reconnecting" type="string">
  `on` or `off` — auto-reconnect on disconnection
</ParamField>

<ParamField body="username" type="string">
  For network cameras
</ParamField>

 <ParamField body="password" type="string">
  For network cameras
</ParamField>


<CodeGroup>

```bash Request
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/connection \
  -H "Content-Type: application/json" \
  -d '{"mode": "remote"}'
```


```json Response
{
  "success": true,
  "message": "Camera connected successfully in remote mode",
  "camera": {
    "connected": true,
    "model": "ILCE-9M3",
    "id": "D10F60149B0C"
  }
}
```

</CodeGroup>

## Disconnect Camera

<ParamField path="method" type="DELETE">
  DELETE /api/cameras/{cameraId}/connection
</ParamField>

Disconnect a camera. Clears event callbacks and waits for clean disconnection.

<CodeGroup>

```bash Request
curl -X DELETE http://localhost:8080/api/cameras/D10F60149B0C/connection
```


```json Response
{
  "success": true,
  "message": "Camera disconnected successfully"
}
```

</CodeGroup>
