## Camera Settings Files

Save and restore complete camera configurations.

### Download Settings to PC

<ParamField path="method" type="POST">`POST /api/cameras/{cameraId}/settings/download`</ParamField>

<ParamField body="filename" type="string">Settings filename (e.g. `CUMSET.DAT`)</ParamField>

```bash
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/settings/download \
  -H "Content-Type: application/json" \
  -d '{"filename": "CUMSET.DAT"}'
```

### Upload Settings to Camera

<ParamField path="method" type="POST">`POST /api/cameras/{cameraId}/settings/upload`</ParamField>

<ParamField body="filename" type="string">Settings filename to restore</ParamField>

```bash
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/settings/upload \
  -H "Content-Type: application/json" \
  -d '{"filename": "CUMSET.DAT"}'
```

### List Settings Files

<ParamField path="method" type="GET">`GET /api/cameras/{cameraId}/settings/files`</ParamField>

```json
{
  "success": true,
  "files": ["CUMSET.DAT", "CUMSET_Backup.DAT"]
}
```
