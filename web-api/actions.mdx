## Shutter

<ParamField path="method" type="POST">`POST /api/cameras/{cameraId}/actions/shutter`</ParamField>

Take a single photo, or control continuous shooting with `down`/`up`.

<ParamField body="action" type="string">
  Optional. `down` to start continuous shooting, `up` to stop. Omit for single shot.
</ParamField>

<CodeGroup>
```bash curl (Single Shot)
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/shutter
```

```bash curl (Continuous Start)
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/shutter \
  -H "Content-Type: application/json" \
  -d '{"action": "down"}'
```

```bash curl (Continuous Stop)
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/shutter \
  -H "Content-Type: application/json" \
  -d '{"action": "up"}'
```

```typescript TypeScript
// Single shot
await client.triggerShutter('D10F60149B0C');

// Continuous start/stop
await client.triggerShutter('D10F60149B0C', { action: 'down' });
await client.triggerShutter('D10F60149B0C', { action: 'up' });
```
</CodeGroup>

<Note>For continuous shooting, set drive mode to a continuous mode first (e.g. `Continuous Hi`).</Note>

## Zoom

<ParamField path="method" type="POST">`POST /api/cameras/{cameraId}/actions/zoom`</ParamField>

Control power zoom lenses. Requires a power zoom lens to be attached.

<ParamField body="direction" type="string" required>`in` or `out`</ParamField>
<ParamField body="speed" type="string" required>`normal` or `fast`</ParamField>

<CodeGroup>
```bash curl
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/zoom \
  -H "Content-Type: application/json" \
  -d '{"direction": "in", "speed": "normal"}'
```

```typescript TypeScript
await client.controlZoom('D10F60149B0C', { direction: 'in', speed: 'normal' });

// Stop zoom
await client.stopZoom('D10F60149B0C');
```
</CodeGroup>

## Focus Near/Far

<ParamField path="method" type="POST">`POST /api/cameras/{cameraId}/actions/focus-near-far`</ParamField>

Move focus in discrete steps. Range: `-7` (near, max speed) to `+7` (far, max speed). Magnitude controls speed. `0` is not valid.

<ParamField body="step" type="integer" required>
  `-7` to `+7`. Negative = near, positive = far.
</ParamField>

<CodeGroup>
```bash curl (Focus Near)
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/focus-near-far \
  -H "Content-Type: application/json" \
  -d '{"step": -1}'
```

```bash curl (Focus Far)
curl -X POST http://localhost:8080/api/cameras/D10F60149B0C/actions/focus-near-far \
  -H "Content-Type: application/json" \
  -d '{"step": 7}'
```

```typescript TypeScript
await client.focusNearFar('D10F60149B0C', { step: -1 }); // near, fine
await client.focusNearFar('D10F60149B0C', { step: 7 });  // far, fast
```
</CodeGroup>

<Tip>
For absolute focus positioning, use `PUT /api/cameras/{id}/properties/focus-position` with a value from 0 (infinity) to 65535 (closest).
</Tip>
