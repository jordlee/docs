## Server Status

<ParamField path="method" type="GET">`GET /api/server/status`</ParamField>

Health check endpoint. Returns server version, uptime, platform, and camera counts. Used by `ServerManager` and `CameraManager` to confirm the server is ready.

<CodeGroup>
```bash curl
curl http://localhost:8080/api/server/status
```

```typescript TypeScript
await manager.isRunning(); // true — uses this endpoint internally
```

```json Response
{
  "success": true,
  "server": {
    "version": "3.0.0",
    "sdkVersion": "V2.01.00",
    "uptime": 42,
    "platform": "darwin-arm64"
  },
  "cameras": {
    "connected": 1,
    "discovered": 2
  }
}
```
</CodeGroup>

## Server Logs

<ParamField path="method" type="GET">`GET /api/server/logs`</ParamField>

Returns buffered server log output. Useful for debugging without terminal access.

<ParamField query="lines" type="integer" default="100">Number of recent log lines</ParamField>
<ParamField query="level" type="string" default="info">`debug`, `info`, `warn`, or `error`</ParamField>

<CodeGroup>
```bash curl
curl http://localhost:8080/api/server/logs
```

```json Response
{
  "success": true,
  "logs": [
    {
      "timestamp": "2026-03-02T12:18:45",
      "level": "info",
      "message": "GET /api/server/status"
    },
    {
      "timestamp": "2026-03-02T12:18:30",
      "level": "info",
      "message": "POST /api/cameras/D10F60149B0C/connection"
    }
  ],
  "total": 47
}
```
</CodeGroup>

## Shutdown

<ParamField path="method" type="POST">`POST /api/server/shutdown`</ParamField>

Initiates graceful shutdown: disconnects all cameras, closes SSE/WebSocket connections, then exits. `CameraManager.close()` and `ServerManager.stop()` call this before falling back to SIGTERM.

```bash
curl -X POST http://localhost:8080/api/server/shutdown
```

```json
{
  "success": true,
  "message": "Shutdown initiated"
}
```
