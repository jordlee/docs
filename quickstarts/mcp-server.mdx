We provide three MCP endpoints that give your AI assistant access to different parts of the camera development ecosystem. Install whichever ones fit your workflow.

| Server | Name | What It Covers |
|--------|------|---------------|
| **SDK Docs** | `CameraRemoteSDK` | Camera Remote SDK (C++/C#) â€” APIs, code examples, compatibility |
| **Help Guides** | `CameraHelp` | Camera help guides for 24 Sony models â€” settings, features, troubleshooting |
| **Web API Docs** | `CameraWebAPI` | Alpha Camera REST API docs â€” endpoints, schemas, examples, and client SDK guidance |

## Choose Your Tool

<Tabs>
  <Tab title="Claude">
    For each server you want to install:

    1. Go to **Settings > Connectors**
    2. Click **Add custom connector**
    3. Enter the name and URL:

    | Name | URL |
    |------|-----|
    | `CameraRemoteSDK` | `https://sdk-rag-agent-production.up.railway.app/mcp` |
    | `CameraHelp` | `https://camera-rag-agent-production.up.railway.app/mcp` |
    | `CameraWebAPI` | `https://crsdk.app/mcp` |

    4. Click **Add**
    5. Open a new chat, click the **search and tools** icon, and toggle the connectors **on**
  </Tab>
  <Tab title="Claude Code">
    ```bash
    # SDK Docs
    claude mcp add CameraRemoteSDK --scope user -- npx mcp-remote https://sdk-rag-agent-production.up.railway.app/mcp

    # Help Guides
    claude mcp add CameraHelp --scope user -- npx mcp-remote https://camera-rag-agent-production.up.railway.app/mcp

    # Web API Docs
    claude mcp add CameraWebAPI --scope user -- npx mcp-remote https://crsdk.app/mcp
    ```

    Verify they're connected:

    ```bash
    claude mcp list
    ```
  </Tab>
  <Tab title="Cursor">
    For each server you want to install:

    1. Open Settings (`Cmd + ,` / `Ctrl + ,`)
    2. Navigate to **Features > MCP**
    3. Click **+ Add New MCP Server**
    4. Set **Transport Type** to `command` (stdio) and enter:

    | Name | Command |
    |------|---------|
    | `CameraRemoteSDK` | `npx mcp-remote https://sdk-rag-agent-production.up.railway.app/mcp` |
    | `CameraHelp` | `npx mcp-remote https://camera-rag-agent-production.up.railway.app/mcp` |
    | `CameraWebAPI` | `npx mcp-remote https://crsdk.app/mcp` |

    5. Click **Save**
  </Tab>
</Tabs>

## Try Your First Query

Ask your AI assistant:

> "How do I connect to a Sony camera and capture a photo using the Camera Remote SDK in C++?"

The AI assistant will automatically search through the relevant MCP server to give you an accurate answer with code examples.

## Example Questions

- "Show me C# code for setting shutter speed" (SDK Docs)
- "What cameras support remote shooting via USB?" (SDK Docs)
- "How do I enable real-time tracking on the A7R V?" (Help Guides)
- "What's the menu path for changing autofocus area on the A7 IV?" (Help Guides)
- "How do I connect using `remote-transfer` and list SD files over the Alpha Camera REST API?" (Web API Docs)
