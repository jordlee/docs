## Product Overview

| Product | What It Does | Language |
|---------|-------------|----------|
| [Client SDK](/sdk/overview) | TypeScript wrapper — managed camera lifecycle, typed APIs, auto-reconnection | TypeScript |
| [HTTP API](/web-api/overview) | RESTful endpoints on localhost — works from any language | Any |
| [Camera Remote SDK](/sdk/camera-remote-sdk) | Sony's official native SDK — the foundation everything else is built on | C++, C# |
| [Camera Remote Command SDK](/sdk/camera-remote-command-sdk) | PTP-based direct camera commands for specialized integrations | C++ |
| [AI SDK Assistant](/MCP-Server/overview) | MCP server connecting AI assistants to 13,000+ pieces of SDK documentation | -- |

## How It All Fits Together

The **Camera Remote SDK** (C++/C#) is the native foundation. The **Camera Remote Web API Server** wraps it as a local HTTP service. The **Client SDK** (TypeScript) manages that server automatically. The **AI SDK Assistant** helps you learn and implement any layer faster.
